# 拖拽功能使用指南

## 🎯 功能說明

您的 Whisper 字幕生成器現在支援真正的檔案拖拽功能！您可以直接從檔案管理器拖拽音訊檔案到應用程式中。

## 🚀 使用方式

### 方法1：拖拽檔案 (推薦)
1. 啟動應用程式：`python main.py --gui`
2. 從檔案管理器中選擇音訊檔案
3. 直接拖拽到應用程式的拖拽區域
4. 看到藍色高亮時放開滑鼠

### 方法2：點擊選擇
1. 點擊拖拽區域
2. 在彈出的檔案選擇對話框中選擇檔案
3. 確認選擇

## 📋 支援格式

- **MP3** - 最常見的音訊格式
- **WAV** - 無壓縮音訊格式
- **M4A** - Apple 音訊格式
- **FLAC** - 無損壓縮格式
- **AAC** - 高效音訊編碼
- **OGG** - 開源音訊格式
- **WMA** - Windows Media 音訊

## 💡 拖拽體驗特色

### 視覺回饋
- **正常狀態**: 灰色邊框，邀請拖拽文字
- **拖拽進入**: 藍色高亮，實線邊框
- **拖拽離開**: 自動恢復正常狀態
- **處理中**: 顯示「正在處理檔案...」

### 批量處理
- 支援一次拖拽多個檔案
- 自動過濾無效檔案
- 顯示檔案數量統計
- 可以分批添加檔案

## 🔧 技術說明

### 拖拽檢測
應用程式會自動檢測拖拽功能是否可用：
- ✅ **完整拖拽支援**: 當 tkinterdnd2 正常工作時
- ⚠️ **降級模式**: 當拖拽不可用時，自動降級為點擊選擇

### 相容性
- **Windows**: 完全支援拖拽
- **macOS**: 可能需要額外設定，會自動降級
- **Linux**: 大部分發行版支援拖拽

## 🛠️ 故障排除

### 拖拽不工作？
如果看到「降級到點擊選擇模式」，表示拖拽功能不可用，但不影響使用：

1. **檢查安裝**:
   ```bash
   pip install tkinterdnd2
   ```

2. **macOS 使用者**:
   - 某些 macOS 版本可能不支援拖拽
   - 使用點擊選擇功能同樣便捷

3. **替代方案**:
   - 點擊拖拽區域選擇檔案
   - 功能完全相同，只是操作方式不同

## 📊 檔案限制

- **最大檔案大小**: 100MB
- **檔案類型檢查**: 自動驗證檔案格式
- **錯誤處理**: 無效檔案會顯示友善提示

## 🎉 最佳實踐

1. **檔案準備**: 確保音訊檔案清晰，背景噪音較少
2. **批量處理**: 一次拖拽多個檔案提高效率
3. **模型選擇**: 根據需要在設定中選擇合適的模型大小
4. **格式選擇**: 推薦使用 SRT 格式獲得最佳相容性
